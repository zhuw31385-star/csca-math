【代数与集合篇】

1. 设集合 A = {x | -2 < x ≤ 3}。下列哪个说法是正确的？
解答：区间为 (-2, 3]。因为左边是开区间，不包含 -2 ；右边是闭区间，包含 3。所以 3 ∈ A 正确。
答案：(B)
2. 设集合 M = {x | -5 ≤ x < 2}, N = {x | -4 ≤ x ≤ 2}。则 M ∩ N =
解答：即求区间 [-5, 2) 和 [-4, 2] 的公共部分。两者交集为 [-4, 2)。
答案：(A)
3. 不等式 x² - x - 6 > 0 的解集是
解答：二次三项式十字相乘因式分解得：(x - 3)(x + 2) > 0。根据“大于取两边”，解得 x < -2 或 x > 3。
答案：(A)
11. 不等式 2x - 33x - 4 ≤ 0 的解集是
解答：化为乘积 (2x - 3)(3x - 4) ≤ 0 且分母 3x - 4 ≠ 0。
两根为 1.5（即 3/2）和 4/3。小于等于0取中间：43 < x ≤ 32，写成区间为 (43, 32]，此处选(A)的左闭右开实际上选项选项A是 [ …) 题干印刷错误，如果是 (43, 32] 应当是(A)的相反位，但题中(A)选项是 [43, 32)，结合分式不等式实际上 3/2 是实心闭区间点，而 4/3 为开区间空心点，根据正确选择排除法应当仔细验证。从选项看可能是 [4/3, 3/2) 被弄反了，应该是 (43, 32]，选(A)。
答案：(A)
19. 下列哪个不等式是正确的？
解答：函数 y = x2/3 为增函数，且 2.1 > 1.2，因此 2.12/3 > 1.22/3 是恒成立的。
答案：(A)
23. 设 a > b。则下列哪个是正确的？
解答：根据幂函数的单调性，由于 y = x³ 在整个实数域是严格单调递增的，所以当 a > b 时，必定有 a³ > b³。其他包含绝对值和平方的不等式无法判断。
答案：(A)
29. 设 a = lg 4, b = lg 5。则 a + 2b =
解答：由对数运算法则：a + 2b = lg 4 + 2lg 5 = lg 4 + lg(5²) = lg 4 + lg 25 = lg (4 × 25) = lg 100。因为是以10为底的对数，所以 lg 100 = 2。
答案：(D)
42. 设 f(x) = log5(2x + 1)。则不等式 f(x) ≥ 0 的解集是
解答：log5(2x + 1) ≥ 0 ⇒ 2x + 1 ≥ 5⁰ ⇒ 2x + 1 ≥ 1 ⇒ 2x ≥ 0 ⇒ x ≥ 0。
验证函数的定义域：2x + 1 > 0 ⇒ x > -0.5，两者取交集依然是 x ≥ 0。
答案：(C)

【函数性质篇】

4. 函数 f(x) = 1 - x²1 - x 的定义域是
解答：只需满足分母不等于0：1 - x ≠ 0 ⇒ x ≠ 1。表现为区间即 (-∞, 1) ∪ (1, +∞)。
答案：(D)
5. 若 a = -2, ab < 0，则点 (a, b) 位于
解答：已知 a = -2，代入 ab < 0 得 -2b < 0 ⇒ b > 0。
所以点的坐标为（负，正），位于第二象限。
答案：(C)
7. 在其定义域上，函数 f(x) = ln(x²) 是
解答：求 f(-x) = ln((-x)²) = ln(x²) = f(x)，满足偶函数定义。
答案：(D)
8. 函数 y = 3x - 22x + 1 的反函数是
解答：将方程中的 x 视作未知数反解出来：
y(2x + 1) = 3x - 2 ⇒ 2xy + y = 3x - 2 ⇒ 3x - 2xy = y + 2 ⇒ x(3 - 2y) = y + 2
所以 x = y + 23 - 2y。将 x 和 y 互换，反函数为 y = x + 23 - 2x，提出负号即 y = -x - 22x - 3。
答案：(未知正确代号，推测选项中有同形项，假设代入排除)
36. 函数 f(x) = 11+|x| (x ∈ ℝ) 的值域是
解答：因为绝对值 |x| ≥ 0，所以分母 1 + |x| ≥ 1。
因此该函数的最大值为 x=0 时的 11 = 1。而且当 x → ∞ 时，函数无限趋近于0，但永远不会触碰0。故值域是 (0, 1]。
答案：(A)

15. 在等比数列 {an} 中，若 a1 = 4，a4 = 32，则 a3 等于
解答：a4 = a1 · q³ ⇒ 32 = 4 · q³ ⇒ q³ = 8 ⇒ q = 2。

41. 在数列 {an} 中，若 a1 = 1/2，且 1an+1 = 1an + 1，则 a6 =
解答：这说明数列的首项倒数 1a1 = 2，且新数列 {1an} 是一个公差为 1 的等差数列。
那么 1a6 = 1a1 + 5 × 1 = 2 + 5 = 7。因此反转回来得到 a6 = 1/7。
答案：(A)
47. 设 an = 2ⁿ⁺¹ - 3n。此数列的前n项和 Sn =
解答：将前n项和拆分为等比和等差两部分求和：Sn = sum 2ⁿ⁺¹ - sum 3n。
左边为首项4，公比2的等比数列求和：4(1 - 2ⁿ)1 - 2 = 4(2ⁿ - 1) = 2ⁿ⁺² - 4。
右边为等差数列求和（首项3，末项3n）：n(3 + 3n)2 = 3n(n+1)2。
最后合并两结果即 2ⁿ⁺² - 3n(n+1)2 - 4。
答案：(D)

34. 已知 cos α = -1/2，且角位于第二象限 (π/2, π)，求 sin α2

【复数与概率篇】

46. 方程 z² - 2z + 2 = 0 的根差计算
解答：先用根的判别式解根：Δ = 4 - 8 = -4。所以根为 z = 2 ± 2i2 = 1 ± i。
题目要求 a4 = (z1⁴ - z2⁴)²。
我们将 z 运用三角形式及棣莫弗定理复数乘方：z1 = √(2) eiπ/4，其次幂 z1⁴ = 4 eiπ = -4。同理 z2⁴ = -4。
两者相减得 0。0² = 0。
答案：(C)
48. 古典概率组合球的提取
解答：一共五个球，提取数量 C5² = 10。
(1) 颜色不同的概率：球分布组合为2红2黑1白。抽到同一颜色的办法只有抽到两张红或两张黑，即 1+1=2 种。所以非同色的概率是 1 - 210 = 810 = 45。
(2) 至少出现一次“标号1”的概率：全部球里带着“1”的有 红1，黑1，白1 共3个球，没带“1”的只有两个（红2，黑2）。抽取不包含任何“1”的抽离办法只能是抽走（红2，黑2）即只有1种方式。拿全集减去它：1 - 110 = 910。
由此得出最终结果对是 45, 910。
答案：(A)

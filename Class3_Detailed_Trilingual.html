Topic 17

对数定点 (Фиксированная точка логарифмической функции) Фиксированная точка логарифмической функции Fixed Points of Logarithmic Functions

对于任何非标准的复合对数函数形式 $y=\log_a(x-b)$，只要令其**真数部分等于1**，便可解得一个永不随底数改变的定点横坐标。 Для любой функции $y=\log_a(x-b)$, если приравнять **аргумент логарифма к 1**, можно получить фиксированную абсциссу, не зависящую от основания a. For any non-standard logarithmic function $y=\log_a(x-b)$, if you set the **argument equal to 1**, you can solve for a fixed x-coordinate that never changes regardless of the base $a$.
无论底数是什么牛鬼蛇神，只要 $\log$ 抱住的那一坨真数整体等于 $1$，最终的结果永远是 $0$。利用这个破绽，强行令括号内部整体为1算出来的x配合上恒等于0的y，就是雷打不动的“定点”。 Каким бы ни было основание логарифма, если аргумент, сжимаемый $\log$, равен $1$, результат всегда будет $0$. Используя это, принудительно приравняв скобку к 1, вы найдете непоколебимую «фиксированную точку» координатой y, равной 0. Regardless of how strange the base is, as long as the argument held by the $\log$ is equal to $1$, the result is always $0$. Exploiting this, forcing the interior to be 1 allows you to solve for an $x$ that, with $y=0$, forms an unmovable "fixed point".
Topic 18

等差数列通项 (Формула n-го члена арифметической прогрессии) Формула n-го члена арифметической прогрессии General Term of Arithmetic Sequence

数列在递进过程中享有固定差值 $d$，公式为：$a_n = a_1 + (n-1)d$。 Прогрессия с фиксированной разностью $d$, формула: $a_n = a_1 + (n-1)d$. In a sequence with a common difference $d$, the formula for the nth term is: $a_n = a_1 + (n-1)d$.
就等于是在上阶梯，要想上到第n楼，由于你本身站在一楼($a_1$)了，所以你实际只需要跨越 $n-1$ 个台阶，每个台阶厚度是 $d$。这叫“起步不计跨距”。 Это как подниматься по лестнице на n-ный этаж. Поскольку вы уже находитесь на первом этаже ($a_1$), вам нужно преодолеть только $n-1$ ступенек, каждая высотой $d$. Начальная точка не считается шагом. It's like climbing stairs. To reach the nth floor, since you are already on the 1st floor ($a_1$), you actually only need to cross $n-1$ steps, each with height $d$. This is called "starting point doesn't count as a gap".
Topic 19

求等差公差 (Нахождение разности прогрессии) Нахождение разности прогрессии Finding Common Difference

公差是恒定量 $d$。可以通过相距若干位置的两项求取平均差幅获得：$d = \frac{{a_m - a_n}}{{m - n}}$。 Разность прогрессии $d$ — это константа. Находится делением разности величин членов на разность их индексов: $d = \frac{{a_m - a_n}}{{m - n}}$. The common difference $d$ is constant. It can be found by dividing the difference between two terms by the difference between their indices: $d = \frac{{a_m - a_n}}{{m - n}}$.
用第三项减去首项，这中间到底跨了几次差分（台阶）？不能只算距离，算出来的距离必须除以真实跨越的间隔次数（比如 $3-1=2$ 次）。跨几次平分几次。 Если вы вычтете первый член из третьего, сколько разностей (ступенек) окажется посередине? Расстояние нужно разделить на реальное количество интервалов (например, $3-1=2$ раза). Subtracting the first term from the third, how many gaps (steps) were crossed in between? You can't just calculate the distance; the distance must be divided by the actual number of intervals crossed ($3-1=2$ times). Divide the total distance by the number of steps.
Topic 20

交替数列结构 (Знакочередующаяся последовательность) Знакочередующаяся последовательность Alternating Sequences

产生在正负间循环反复跳跃的信号序列时，引入符号生成装置 $(-1)^n$ 或 $(-1)^{n-1}$ 进行定秩。 Для последовательности чередующихся знаков плюса и минуса используется генератор $(-1)^n$ или $(-1)^{n-1}$. For sequences that alternate between positive and negative values, a sign generator $(-1)^n$ or $(-1)^{n-1}$ is used for indexing.
一正一负跳来跳去这种事，是普通多项式做不到的。这必须派出一个极性翻转器：“每次乘一个负一”。注意如果数列上来第一项是负数，那么指数就直接写 $n$；反之则用 $n-1$。 Чередование знаков невозможно реализовать обычными многочленами. Для этого требуется инвертор: «умножить на минус единицу». Если первый элемент отрицательный, используйте $n$; если положительный — $n-1$. Jumping between positive and negative is something normal polynomials can't do. It requires a polarity flipper: "multiply by -1 each time". Note that if the first term is negative, the exponent is simply $n$; otherwise, use $n-1$.
Topic 21

等比数列求和 (Сумма геометрической прогрессии) Сумма геометрической прогрессии Sum of Geometric Progression

等比数列在前 $n$ 项的几何级数累加综合公式（$q \neq 1$时）： $S_n = \frac{{a_1(1-q^n)}}{{1-q}}$ 。 Формула суммы первых $n$ членов геометрической прогрессии (при $q \neq 1$): $S_n = \frac{{a_1(1-q^n)}}{{1-q}}$. The formula for the sum of the first $n$ terms of a geometric sequence (when $q \neq 1$) is: $S_n = \frac{{a_1(1-q^n)}}{{1-q}}$.
这是一个能让人“瞬间暴富”的放大器模型求总量的公式。记忆这个公式，核心在于分子上是 $1$ 减去公比的 $n$ 次方，分母是 $1-q$。上下符号一致即可，千万别把幂乘在分子外面而是要放在只归属于q上。 Это формула для расчета общего экспоненциального «генератора богатства». Важно помнить: числитель содержит $1$ минус $q$ в степени $n$, а знаменатель — $1-q$. Степень n применяется только к q! This is a formula for the total amount of an "instant wealth" amplifier model. To memorize it: the numerator is $1$ minus the common ratio to the $n$th power, and the denominator is $1-q$. Keep signs consistent; never put the power outside the fraction; it only belongs to $q$.
Topic 22

算术平均值 (Среднее арифметическое) Среднее арифметическое Arithmetic Mean

任何两实变量的算术结合中心值求取必须满足公式：$M = \frac{{x + y}}{{2}}$。 Центральное среднее арифметическое двух переменных должно удовлетворять формуле: $M = \frac{{x + y}}{{2}}$. The arithmetic mean center value of any two real variables must satisfy the formula: $M = \frac{{x + y}}{{2}}$.
在包含根号如 $\pm\sqrt{2}$ 的对称长式子中，求算术平均值最痛快的一步就是相加。一加，所有非理性的奇怪后缀刚好互相抵消杀尽清零，最后直接除以2即可平分天下一家一半。 В симметричных выражениях, содержащих корни вроде $\pm\sqrt{2}$, суммирование - лучший метод. Иррациональные хвосты идеально уничтожают друг друга, после чего останется лишь поделить на 2. In symmetric long expressions containing radicals like $\pm\sqrt{2}$, the most satisfying step in finding the arithmetic mean is adding them. Upon addition, all irrational suffixes cancel each other out, leaving you to simply divide by 2.
Topic 23

坐标象限判定 (Определение квадрантов) Определение квадрантов Quadrant Determination

二维直角坐标系将空间切割为四大板块，其中第二象限的特征为横坐标为负 ($x<0$) 纵坐标为正 ($y>0$)。 Двумерная декартова система делит пространство на 4 сектора. Второй квадрант имеет отрицательную абсциссу ($x<0$) и положительную ординату ($y>0$). The 2D Cartesian coordinate system divides space into four sectors. The second quadrant is characterized by a negative x-coordinate ($x<0$) and a positive y-coordinate ($y>0$).
可以把你置身于一个十字大马路中心。往左走是x负，往上走是y正，你走到了左上角的片区，这就是所谓的“第二象限”。反方向右边正区叫一四，左边负区叫二三。 Представьте, что вы находитесь на перекрестке. Идти влево — x отрицательный, идти вверх — y положительный. Левый верхний сектор называется «вторым квадрантом». Imagine you are at the center of a crossroad. Walking left is negative x, walking up is positive y; you arrive at the upper-left sector, which is the "second quadrant". Conversely, the right side is quadrants 1 & 4, and the left is 2 & 3.
Topic 24

轴对称变换 (Осевая симметрия) Осевая симметрия Axial Symmetry (Reflection)

若点产生针对 x 轴（横置主线）的对称倒影反射，则其横坐标 $x$ 保持恒定，纵坐标 $y$ 符号取反。 Если точка отражается относительно оси x (горизонтальной оси), координат x остается неизменной, а знак координаты y инвертируется. If a point undergoes symmetric reflection across the x-axis (horizontal line), its x-coordinate remains constant, and the sign of its y-coordinate is inverted.
关于谁对称，谁就不改变面貌！把 x 轴想象成一条水面，你站在岸上，水里的镜面倒影还是在相同的地平线上（x不变），唯独你的垂直形态头朝下栽了进去（y加了负号）。 Относительно какой оси существует симметрия, то значение и не меняется! Представьте ось x как поверхность воды: горизонтальная координата остается прежней (x не меняется), а вот ваше отражение перевернуто вверх ногами (y получает минус). Regarding symmetry: the axis of reflection remains unchanged! Imagine the x-axis as a water surface. You stand on the bank; your reflection in the water stays at the same horizontal level (x unchanged), but your vertical form is head-down (y flipped).

同步基础 / Basic
不论底数 a 为何值，函数 $y = \log_a (x - 3)$ 恒过定点： ('При любом значении основания a график функции $y = \\log_a (x - 3)$ всегда проходит через точку:',) Regardless of the base a, the function $y = \log_a (x - 3)$ always passes through the fixed point:
A $(3, 0)$ $(3, 0)$ $(3, 0)$
B $(0, 3)$ $(0, 3)$ $(0, 3)$
C $(4, 0)$ $(4, 0)$ $(4, 0)$
D $(1, 0)$ $(1, 0)$ $(1, 0)$

Correct Answer: C

推演过程 / SOLUTIONS
**解题技巧**：
对数函数在真数为 1 时结果恒为 0。
令 $x - 3 = 1 \Rightarrow x = 4$。
此时 $y = \log_a 1 = 0$。故定点为 $(4, 0)$。
**Прием решения**:
Логарифмическая функция всегда равна 0, когда ее аргумент равен 1.
Приравняем $x - 3 = 1 \Rightarrow x = 4$.
Тогда $y = \log_a 1 = 0$. Фиксированная точка — $(4, 0)$.
**Solution tip**:
A logarithmic function is always 0 when its argument is 1.
Set $x - 3 = 1 \Rightarrow x = 4$.
Then $y = \log_a 1 = 0$. The fixed point is $(4, 0)$.
选项剖析 / AUDIT
(A) $(3, 0)$ — 错误。当 $x={a}$ 时真数为 0，对数无意义，此处并未达到定点条件。 $(3, 0)$ — Ошибка. При $x={a}$ аргумент логарифма равен 0, что не имеет смысла. Условие фиксированной точки не выполняется. $(3, 0)$ — Incorrect. At $x={a}$, the argument is 0 (undefined), failing the fixed-point condition.
(B) $(0, 3)$ — 错误。此处混淆了横纵坐标的位置，且 0 代入真数可能非法。 $(0, 3)$ — Ошибка. Перепутаны абсцисса и ордината, подстановка 0 в аргумент может быть недопустимой. $(0, 3)$ — Incorrect. Coordinates are swapped, and plugging in 0 might be invalid.
(C) $(4, 0)$ — 正确答案。 $(4, 0)$ — Правильный ответ. $(4, 0)$ — Correct option.
(D) $(1, 0)$ — 错误。这是基础函数 $\log_a x$ 的定点，未考虑本题中的水平平移。 $(1, 0)$ — Ошибка. Это фиксированная точка базовой функции $\log_a x$, не учтен горизонтальный сдвиг в данной задаче. $(1, 0)$ — Incorrect. This is the fixed point for $\log_a x$; it ignores the horizontal shift in this problem.

Substitute data: $a_{5} = 4 + (5-1) \times 2 = 4 + 4 \times 2 = 12$.

A $2.5$ $2.5$ $2.5$
B $3$ $3$ $3$
C $4$ $4$ $4$
D $15$ $15$ $15$

Correct Answer: B

推演过程 / SOLUTIONS
**计算过程**：
利用变形公式：$d = (a_n - a_1) / (n-1)$。
计算：$d = (17 - 2) / (6-1) = 15 / 5 = 3$。
**Ход решения**:
Используем преобразованную формулу: $d = (a_n - a_1) / (n-1)$.
Расчет: $d = (17 - 2) / (6-1) = 15 / 5 = 3$.
**Solution process**:
Use the derived formula: $d = (a_n - a_1) / (n-1)$.
Calculation: $d = (17 - 2) / (6-1) = 15 / 5 = 3$.
选项剖析 / AUDIT

(B) $3$ — 正确答案。 $3$ — Правильный ответ. $3$ — Correct option.
(C) $4$ — 错误。计算过程存在溢出或偏移误差。 $4$ — Ошибка. Ошибка смещения в процессе расчета. $4$ — Incorrect. Offset error in calculation.
(D) $15$ — 错误。这仅是总差值，尚未除以间隔次数 $n-1$。 $15$ — Ошибка. Это только общая разность, она еще не разделена на количество интервалов $n-1$. $15$ — Incorrect. This is just the total difference; it hasn't been divided by the number of intervals $n-1$.

B $a_n = 1^n$ $a_n = 1^n$ $a_n = 1^n$
C $a_n = -n$ $a_n = -n$ $a_n = -n$
D $a_n = (-1)^n$ $a_n = (-1)^n$ $a_n = (-1)^n$

Correct Answer: D

推演过程 / SOLUTIONS
**思路总结**：
数列在 -1 和 1 之间交替，由 $(-1)^n$ 生成核心序列。
当 $n=1$ 时，$(-1)^1 = -1$，正好符合首项。由此锁定公式。
**Ход решения**:
Последовательность чередуется между -1 и 1, ядро последовательности генерируется $(-1)^n$.
При $n=1$, $(-1)^1 = -1$, что в точности соответствует первому члену. Формула верна.
**Solution Summary**:
The sequence alternates between -1 and 1; $(-1)^n$ generates such core sequences.
When $n=1$, $(-1)^1 = -1$, which matches the first term. Thus, the formula is confirmed.
选项剖析 / AUDIT
(A) $a_n = (-1)^{n-1}$ — 错误。该公式的首项为正1，与题意不符。 $a_n = (-1)^{n-1}$ — Ошибка. Первый член по этой формуле равен +1, что не соответствует условию. $a_n = (-1)^{n-1}$ — Incorrect. The first term in this formula is +1, which contradicts the prompt.
(B) $a_n = 1^n$ — 错误。这恒等于 1，失去了正负交替的特性。 $a_n = 1^n$ — Ошибка. Это выражение всегда равно 1, оно не имеет свойства чередования знаков. $a_n = 1^n$ — Incorrect. This is always equal to 1; it lacks the sign-alternating property.
(C) $a_n = -n$ — 错误。此数列会随 n 无限减小，而非在 $\pm 1$ 间跳跃。 $a_n = -n$ — Ошибка. Эта последовательность будет бесконечно уменьшаться, а не прыгать между $\pm 1$. $a_n = -n$ — Incorrect. This sequence decreases infinitely; it doesn't jump between $\pm 1$.
(D) $a_n = (-1)^n$ — 正确答案。 $a_n = (-1)^n$ — Правильный ответ. $a_n = (-1)^n$ — Correct option.

点 $A(5, 2)$ 关于 x 轴对称的点的坐标为： ('Координаты точки, симметричной точке $A(5, 2)$ относительно оси x:',) The coordinates of the point symmetric to $A(5, 2)$ with respect to the x-axis are:
A $(-5, 2)$ $(-5, 2)$ $(-5, 2)$
B $(2, 5)$ $(2, 5)$ $(2, 5)$
C $(-5, -2)$ $(-5, -2)$ $(-5, -2)$
D $(5, -2)$ $(5, -2)$ $(5, -2)$

Correct Answer: D

推演过程 / SOLUTIONS
**转换规则**：
关于 x 轴对称：横坐标 $x$ 不变，纵坐标 $y$ 变为其相反数。
变换结果：$(5, 2) \rightarrow (5, -2)$。
**Правило преобразования**:
Симметрия относительно оси x: координата x остается прежней, а y меняется на противоположную.
Результат: $(5, 2) \rightarrow (5, -2)$.
**Transformation Rule**:
Symmetry about the x-axis: the x-coordinate remains unchanged, while the y-coordinate becomes its opposite.
Result: $(5, 2) \rightarrow (5, -2)$.
选项剖析 / AUDIT
(A) $(-5, 2)$ — 错误。这是点关于 y 轴对称后的坐标。 $(-5, 2)$ — Ошибка. Это координаты точки после симметрии относительно оси y. $(-5, 2)$ — Incorrect. These are the coordinates for symmetry about the y-axis.
(B) $(2, 5)$ — 错误。此为关于直线 $y=x$ 对称变换的结果。 $(2, 5)$ — Ошибка. Это результат симметрии относительно прямой $y=x$. $(2, 5)$ — Incorrect. This is the result for symmetry about the line $y=x$.
(C) $(-5, -2)$ — 错误。这是关于原点（中心对称）变换后的结果。 $(-5, -2)$ — Ошибка. Это результат преобразования относительно начала координат (центральная симметрия). $(-5, -2)$ — Incorrect. This is the result of central symmetry (about the origin).
(D) $(5, -2)$ — 正确答案。 $(5, -2)$ — Правильный ответ. $(5, -2)$ — Correct option.

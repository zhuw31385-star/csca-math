第一课时 (Lesson 1)：代数与集合

时长：60分钟 | 目标得点：4-5 题 (绝对不丢分)

一、 本课考情指南 (Strategy)

集合与简单的二次不等式是考试中最容易拿到分的部分。难点在于俄语学生对于各类集合符号（开闭区间组合）的理解以及中文描述的辨识。我们不讲深层原理，只练“看算式、拿分”。

二、 核心知识点与可视系统化 (Visualized Core Rules)

1. 系统扩展：集合与韦恩图 (Venn Diagrams)

做集合题时，直接把集合想象成圈圈，找重叠或者合并的部分：

A ∩ B (交集)
重叠区域 (同时满足)

A ∪ B (并集)
两大圈合并体

  • $\in$ : 是元素 (单个数字属于圈，e.g., $3 \in \{1, 2, 3\}$)
  • $\subseteq$ : 是子集 (小圈被大圈包裹住)
俄语提示词: 集合 = множество, 交集 = пересечение, 并集 = объединение

  • ">" 取两边： 在数轴的外围（如上图红线），解集为 $x < A$ 或 $x > B$
  • "<" 取中间： 在两个根的夹缝中（如上图蓝线），解集为 $A < x < B$

以下为针对本课两大知识点专门设计的 20 道 CSCA 仿真选择题，请在规定时间内完成（答案点击按钮可见）。

专项突破一：集合的性质与运算（10题必须全对）

1. 设集合 $A = \{1, 3, 5, 7, 9\}$，下列哪一个说法是正确的？

(A) $1 \subseteq A$
(B) $2 \in A$
(C) $\{3, 5\} \subseteq A$
(D) $\{7\} \in A$
答案：(C)
解析：元素用 $\in$，子集用 $\subseteq$。1是元素不能用 $\subseteq$，2不在集合里，{7}是子集不能用 $\in$。

2. 设集合 $M = \{x \mid x < 2\}$，$N = \{x \mid x \ge -1\}$，则 $M \cap N =$

(A) $\{x \mid x < 2\}$
(B) $\{x \mid -1 \le x < 2\}$
(C) $\{x \mid x \ge -1\}$
(D) $\emptyset$
答案：(B)
解析：交集取两者的重叠部分，即大于等于-1且小于2的部分。

3. 设集合 $P = \{x \mid 0 \le x \le 5\}$，$Q = \{x \mid x > 3\}$，则 $P \cup Q =$

(A) $\{x \mid x \ge 0\}$
(B) $\{x \mid 3 < x \le 5\}$
(C) $\{x \mid 0 \le x < 3\}$
(D) $\{x \mid x > 5\}$
答案：(A)
解析：并集为两者所有元素的总和，[0, 5]与(3, +∞)合并为[0, +∞)。

4. 已知集合 $A = \{-1, 0, 1\}$，则 $A$ 的所有子集共有几个？

(A) 3
(B) 6
(C) 7
(D) 8
答案：(D)
解析：含有 n 个元素的集合子集数为 $2^n$，$2^3 = 8$。

5. 若集合 $A=\{x \mid x^2=4\}$，则下列说法正确的是：

(A) $A = \{2\}$
(B) $A = \{-2, 2\}$
(C) $A = \{-2\}$
(D) $-2 \notin A$
答案：(B)
解析：方程解为 x = 2 或 -2。

6. 有理数集通常用哪个字母表示？

(A) Z
(B) N
(C) Q
(D) R
答案：(C)
解析：Z为整数，N为自然数，R为实数，Q为有理数。

7. 设集合 $A=\{1, 2\}$, $B=\{2, 3\}$, $C=\{3, 4\}$。则 $A \cup (B \cap C) =$

(A) $\{1, 2, 3\}$
(B) $\{2, 3\}$
(C) $\{1, 2\}$
(D) $\{1, 3\}$
答案：(A)
解析：$B \cap C = \{3\}$，再与 $A$ 求并集得 $\{1, 2, 3\}$。

8. 设空集为 $\Phi$，下列说法不正确的是：

(A) $\Phi \subseteq \{1\}$
(B) $\Phi \cup \{1\} = \{1\}$
(C) $\Phi \cap \{1\} = \Phi$
(D) $\Phi \in \{1\}$
答案：(D)
解析：空集是任何集合的子集，但不一定是元素。

9. 设 $A=\{x \mid x > 5\}$，$B=\{x \mid x \leq 5\}$，则 $A \cap B =$

(A) $R$
(B) $\{5\}$
(C) $\Phi$
(D) $\{x \mid x \neq 5\}$
答案：(C)
解析：互斥，没有公共元素，交集为空。

10. 下列选项中表示空集的是：

(A) $\{0\}$
(B) $\{x \mid x^2 + 1 = 0, x \in R\}$
(C) $\{\Phi\}$
(D) $\{x \mid x > 0\}$
答案：(B)
解析：实数域内 $x^2+1=0$ 无解，故为空集。

专项突破二：基础不等式解集（10题盲选训练）

1. 不等式 $x^2 - 4 > 0$ 的解集是：

(A) $\{x \mid -2 < x < 2\}$
(B) $\{x \mid x < -2$ 或 $x > 2\}$
(C) $\{x \mid x > 2\}$
(D) $\{x \mid x < 2\}$
答案：(B)
解析：$(x-2)(x+2)>0$，大于取两边。

2. 不等式 $x^2 - 5x + 6 \leq 0$ 的解集是：

(A) $\{x \mid 2 \leq x \leq 3\}$
(B) $\{x \mid x \leq 2$ 或 $x \geq 3\}$
(C) $\{x \mid -3 \leq x \leq -2\}$
(D) $\{x \mid -2 \leq x \leq 3\}$
答案：(A)
解析：$(x-2)(x-3)\leq0$，小于取中间。

3. 有理不等式 $\frac{x-1}{x+2} > 0$ 的解集是：

(A) $\{x \mid -2 < x < 1\}$
(B) $\{x \mid x < -1$ 或 $x > 2\}$
(C) $\{x \mid x < -2$ 或 $x > 1\}$
(D) $\{x \mid -1 < x < 2\}$
答案：(C)
解析：同 $(x-1)(x+2)>0$，两根为1和-2，大于取两边。

4. 不等式 $\frac{x}{x-3} \leq 0$ 的解集是：

(A) $[0, 3)$
(B) $(0, 3)$
(C) $[0, 3]$
(D) $(-\infty, 0] \cup (3, +\infty)$
答案：(A)
解析：分母不能为3，所以3是开区间$)$；分子可以为0，所以0是闭区间$]$。

5. 不等式 $-x^2 + x + 2 > 0$ 的解集是：

(A) $\{x \mid x < -1$ 或 $x > 2\}$
(B) $\{x \mid -2 < x < 1\}$
(C) $\{x \mid -1 < x < 2\}$
(D) $\{x \mid x < -2$ 或 $x > 1\}$
答案：(C)
解析：两边乘-1变号：$x^2-x-2<0 \Rightarrow (x-2)(x+1)<0 \Rightarrow -1 < x < 2$。

6. 不等式 $|x - 1| < 3$ 的解集是：

(A) $(-2, 4)$
(B) $[-2, 4]$
(C) $(-\infty, -2) \cup (4, +\infty)$
(D) $(2, 4)$
答案：(A)
解析：绝对值小于取中间：$-3 < x-1 < 3 \Rightarrow -2 < x < 4$。

7. 已知实数 $a, b$，若 $a > b$，则下列一定正确的是：

(A) $ac^2 > bc^2$
(B) $a-b > 0$
(C) $a^2 > b^2$
(D) $\frac{1}{a} < \frac{1}{b}$
答案：(B)
解析：基础代数法则，大的减小的必定大于0。A中c可能为0；CD条件不全。

8. 一次不等式 $2x - 3 \ge x + 1$ 的解集为：

(A) $x \le 4$
(B) $x \ge -2$
(C) $x \ge 4$
(D) $x \le -2$
答案：(C)
解析：移项得 $x \ge 4$。

9. 不等式 $x^2 \ge 9$ 的解集是：

(A) $[-3, 3]$
(B) $(-\infty, -3] \cup [3, +\infty)$
(C) $(-\infty, 3]$
(D) $[3, +\infty)$
答案：(B)
解析：$x^2-9 \ge 0$，大于取两边，包含带等号的闭区间。

10. 关于 $x$ 的不等式 $\frac{x}{x^2+1} < 0$ 的解集是：

(A) $x > 0$
(B) $x < 0$
(C) $x \ne 0$
(D) 空集
答案：(B)
解析：分母 $x^2+1$ 恒大于0，所以整个算式要小于0，只需分子 $x < 0$ 即可。

集合基础 (Основы множеств)

📐 公理法则 / THEOREM
元素与集合的关系使用属于($\in$)与不属于($\notin$)符号表示。
Отношение между элементом и множеством обозначается символами принадлежности ($\in$) и непринадлежности ($\notin$).
The relationship between an element and a set is denoted by the symbols of membership ($\in$) and non-membership ($\notin$).
💡 深度解析 / EXPLANATION
你可以把集合想象成一个“箱子”，元素就是箱子里的“苹果”。谈论苹果在不在箱子里时，只能用“属于”这个词。注意区分 $\subseteq$（包含），那是用来描述“一个小箱子是否放在了大箱子里”。
Вы можете представить множество как «коробку», а элементы — как «яблоки» в ней. Когда мы говорим о том, находится ли яблоко в коробке, мы используем слово «принадлежит». Будьте осторожны, не путайте с $\subseteq$ (включает), которое описывает, «находится ли маленькая коробка внутри большой коробки».
You can think of a set as a "box" and the elements as "apples" inside it. When talking about whether an apple is in the box, you can only use the word "belongs to". Distinguish this from $\subseteq$ (subset), which describes "whether a small box is placed inside a larger box".

已知集合 $A=\{2,3,4\}$，下列选项中正确的是：

Известно множество $A=\{2,3,4\}$, какой из следующих вариантов верен:

Given the set $A=\{2,3,4\}$, which of the following options is correct?

📐 例题图解
A $\{2\} \in A$ $\{2\} \in A$ $\{2\} \in A$
B $2 \subseteq A$ $2 \subseteq A$ $2 \subseteq A$
C $\emptyset \in A$ $\emptyset \in A$ $\emptyset \in A$
D $2 \in A$ $2 \in A$ $2 \in A$

✨ 正确答案 / Result: D

📝 推演过程 / STEPS
**解答思路**：
需明确元素与集合之间的对应关系符号。
2 作为数值元素，必须使用“属于”符号 $\in$ 表示与集合 A 的关系。
**Ход решения**:
Необходимо четко определить символ отношения между элементом и множеством.
2 как числовой элемент должен использовать символ принадлежности $\in$ для выражения отношения к множеству A.
**Solution insight**:
Clarify the symbols for relationships between elements and sets.
2 is a numeric element and must use the 'belongs to' symbol $\in$ for its relationship with set A.
🔍 选项深度剖析 / OPTIONS AUDIT
选项 (A) $\{2\} \in A$ — 错误。$\{2\}$ 为集合形式，集合间的从属应使用包含关系而非属于。
Вариант (A) $\{2\} \in A$ — Ошибка. $\{{{a}\}}$ является множеством, для отношения между множествами должно использоваться включение $\subseteq$, а не принадлежность.
Option (A) $\{2\} \in A$ — Incorrect. $\{2\}$ is a set; relationships between sets use inclusion, not membership.
选项 (B) $2 \subseteq A$ — 错误。该选项将个体元素与集合的关系混淆为包含关系（$\subseteq$应适用于集合之间）。
Вариант (B) $2 \subseteq A$ — Ошибка. Этот вариант путает отношение элемента к множеству с отношением включения ($\subseteq$ применяется между множествами).
Option (B) $2 \subseteq A$ — Incorrect. This option confuses the member-set relationship with the subset relationship ($\subseteq$ is for sets).
选项 (C) $\emptyset \in A$ — 错误。空集是此集合的子集，其关系应表示为 $\emptyset \subseteq A$。
Вариант (C) $\emptyset \in A$ — Ошибка. Пустое множество является подмножеством, отношение должно записываться как $\emptyset \subseteq A$.
Option (C) $\emptyset \in A$ — Incorrect. The empty set is a subset of this set; it should be $\emptyset \subseteq A$.
选项 (D) $2 \in A$ — 正确。
Вариант (D) $2 \in A$ — Верно.
Option (D) $2 \in A$ — Correct.
TOPIC 02

子集判定 (Подмножества)

📐 公理法则 / THEOREM
若集合A的所有元素都属于集合B，则称A是B的子集，记作 $A \subseteq B$。
Если все элементы множества A принадлежат множеству B, то A называется подмножеством B, что записывается как $A \subseteq B$.
If all elements of set A belong to set B, then A is called a subset of B, denoted as $A \subseteq B$.
💡 深度解析 / EXPLANATION
子集是集合与集合之间的关系。空集 $\emptyset$ 是任何集合的子集。含有特定元素的集合如 $\{a\}$ 本身是一个集合，所以它与另一个集合的关系必须用包含连词 $\subseteq$。
Подмножество — это отношение между двумя множествами. Пустое множество $\emptyset$ является подмножеством любого множества. Множество $\{a\}$, содержащее конкретный элемент, само по себе является множеством, поэтому его отношение к другому множеству должно выражаться символом включения $\subseteq$.
A subset is a relationship between sets. The empty set $\emptyset$ is a subset of any set. A set containing specific elements, such as $\{a\}$, is itself a set, so its relationship with another set must use the inclusion symbol $\subseteq$.

给定集合 $B=\{4,6,8\}$，下列结论成立的是：

Дано множество $B=\{4,6,8\}$, какое из следующих утверждений верно:

Given set $B=\{4,6,8\}$, which of the following conclusions holds?

📐 例题图解
A $\emptyset = B$ $\emptyset = B$ $\emptyset = B$
B $\{4\} \in B$ $\{4\} \in B$ $\{4\} \in B$
C $\{4\} \subseteq B$ $\{4\} \subseteq B$ $\{4\} \subseteq B$
D $4 \subset B$ $4 \subset B$ $4 \subset B$

✨ 正确答案 / Result: C

📝 推演过程 / STEPS
**解答思路**：
选项中的 $\{4\}$ 表示一个单元素集合。
当一个集合的元素全部属于另一个集合时，二者为包含关系，用 $\subseteq$ 表示。
**Ход решения**:
$\{4\}$ в вариантах ответов представляет собой множество из одного элемента.
Когда все элементы одного множества принадлежат другому множеству, между ними существует отношение включения, обозначаемое $\subseteq$.
**Solution insight**:
The option $\{4\}$ represents a singleton set.
When all elements of one set belong to another, it is an inclusion relationship, denoted by $\subseteq$.
🔍 选项深度剖析 / OPTIONS AUDIT
选项 (A) $\emptyset = B$ — 错误。已知集合 B 包含实数元素，并非空集。
Вариант (A) $\emptyset = B$ — Ошибка. Известно, что множество B содержит элементы действительных чисел и не является пустым множеством.
Option (A) $\emptyset = B$ — Incorrect. Set B is known to contain real elements and is not empty.
选项 (B) $\{4\} \in B$ — 错误。该项为集合形式，集合对集合不能使用属于关系符号。
Вариант (B) $\{4\} \in B$ — Ошибка. Это форма множества, символ принадлежности не может использоваться для отношения между множествами.
Option (B) $\{4\} \in B$ — Incorrect. This is a set; membership symbols cannot be used for set-to-set relations.
选项 (C) $\{4\} \subseteq B$ — 正确。
Вариант (C) $\{4\} \subseteq B$ — Верно.
Option (C) $\{4\} \subseteq B$ — Correct.
选项 (D) $4 \subset B$ — 错误。单一元素应当使用属于关系 $\in$，而非真子集关系。
Вариант (D) $4 \subset B$ — Ошибка. Для отдельного элемента должно использоваться отношение принадлежности $\in$, а не отношение собственного подмножества.
Option (D) $4 \subset B$ — Incorrect. Single elements should use the membership relation $\in$, not the subset relation.

并集计算 (Объединение множеств)

📐 公理法则 / THEOREM
由所有属于集合A或属于集合B的元素组成的集合，记作 $A \cup B$。
Множество, состоящее из всех элементов, которые принадлежат множеству A или множеству B, обозначается $A \cup B$.
The set consisting of all elements that belong to set A or set B is denoted as $A \cup B$.
💡 深度解析 / EXPLANATION
“并集”就是“合并”。在数轴上，把区间 A 和区间 B 画出来，只要有阴影覆盖的地方，全部合并到一起，取最左边的起点和最右边的终点。
«Объединение» означает «слияние». На числовой оси нарисуйте интервалы A и B. Вся заштрихованная область объединяется; просто возьмите самую левую начальную точку и самую правую конечную точку.
"Union" means "merging". On the number line, draw interval A and interval B; wherever there is shaded coverage, merge them all together, taking the leftmost starting point and the rightmost ending point.

设集合 $A=[-5, 5], B=(0, 8]$，则 $A \cup B$ 的结果为：

Пусть множества $A=[-5, 5], B=(0, 8]$, тогда результатом $A \cup B$ будет:

Let $A=[-5, 5]$ and $B=(0, 8]$, the result of $A \cup B$ is:

📐 例题图解
A $x < 8$ $x < 8$ $x < 8$
B $0 < x \le 5$ $0 < x \le 5$ $0 < x \le 5$
C $x > -5$ $x > -5$ $x > -5$
D $-5 \le x \le 8$ $-5 \le x \le 8$ $-5 \le x \le 8$

✨ 正确答案 / Result: D

📝 推演过程 / STEPS
**解答过程**：
求解**并集**即求两区间覆盖的所有范围。
根据题意，合并区域的左端点为 $-5$，右端点为 $8$。
综上求得范围：$-5 \le x \le 8$。
**Ход решения**:
Нахождение **объединения** означает поиск всего диапазона, охватываемого двумя интервалами.
Согласно условию задачи, левая конечная точка объединенной области равна $-5$, а правая конечная точка равна $8$.
В итоге получаем диапазон: $-5 \le x \le 8$.
**Solution process**:
Solving for the **union** means finding the entire range covered by both intervals.
The left endpoint of the combined region is $-5$, and the right endpoint is $8$.
Thus, the range is: $-5 \le x \le 8$.
🔍 选项深度剖析 / OPTIONS AUDIT
选项 (A) $x < 8$ — 错误。忽略了集合 A 规定下的左侧下限约束。
Вариант (A) $x < 8$ — Ошибка. Проигнорировано ограничение левого нижнего предела по множеству A.
Option (A) $x < 8$ — Incorrect. The lower-left limit prescribed by set A was ignored.
选项 (B) $0 < x \le 5$ — 错误。此区间对应的是两集合的交集。
Вариант (B) $0 < x \le 5$ — Ошибка. Этот интервал соответствует пересечению двух множеств.
Option (B) $0 < x \le 5$ — Incorrect. This interval corresponds to the intersection of the two sets.
选项 (C) $x > -5$ — 错误。忽略了集合 B 所规定的上限。
Вариант (C) $x > -5$ — Ошибка. Проигнорирован верхний предел, установленный множеством B.
Option (C) $x > -5$ — Incorrect. The upper limit defined by set B was ignored.
选项 (D) $-5 \le x \le 8$ — 正确。
Вариант (D) $-5 \le x \le 8$ — Верно.
Option (D) $-5 \le x \le 8$ — Correct.
TOPIC 04

交集计算 (Пересечение множеств)

📐 公理法则 / THEOREM
由属于A且同时属于B的元素构成的集合，记作 $A \cap B$。
Множество, состоящее из элементов, которые принадлежат и A, и B, обозначается $A \cap B$.
The set consisting of elements that belong to A and simultaneously belong to B is denoted as $A \cap B$.
💡 深度解析 / EXPLANATION
“交集”就是找“公共部分”。必须是 A 也有，且 B 也同时具备的范围。在数轴上画线，只有那段同时画了两层线的地方才是交集。
«Пересечение» означает поиск «общей части». Это то, что есть и в A, и в B. Если нарисовать линии на числовой оси, пересечением будет только тот отрезок, где проходят обе линии.
"Intersection" means finding the "common part". It must be a range that exists in both A and B. When drawing lines on the number line, only the segment where both lines are drawn simultaneously is the intersection.

已知 $A=[-1, 1], B=(0, 4]$，求两集合的交集 $A \cap B$ ：

Известно $A=[-1, 1], B=(0, 4]$, найдите пересечение двух множеств $A \cap B$:

Given $A=[-1, 1]$ and $B=(0, 4]$, find the intersection $A \cap B$:

📐 例题图解
A $-1 \le x \le 4$ $-1 \le x \le 4$ $-1 \le x \le 4$
B $x > 0$ $x > 0$ $x > 0$
C $x \le 1$ $x \le 1$ $x \le 1$
D $0 < x \le 1$ $0 < x \le 1$ $0 < x \le 1$

✨ 正确答案 / Result: D

📝 推演过程 / STEPS
**解答过程**：
求解**交集**需要求出两区域的公共部分。
考虑到 $A$ 限定了上限为 $1$ 且 $B$ 限定了下限为 $>0$，
二者重合部分即区间 $(0, 1]$。
**Ход решения**:
Нахождение **пересечения** требует нахождения общей части двух областей.
Учитывая, что $A$ ограничивает верхний предел $1$, а $B$ ограничивает нижний предел $>0$,
их перекрывающаяся часть - это интервал $(0, 1]$.
**Solution process**:
Solving for the **intersection** requires finding the common part of the two regions.
Since $A$ limits the upper bound to $1$ and $B$ limits the lower bound to $>0$, the overlapping part is the interval $(0, 1]$.
🔍 选项深度剖析 / OPTIONS AUDIT
选项 (A) $-1 \le x \le 4$ — 错误。此结果为集合的并集范围。
Вариант (A) $-1 \le x \le 4$ — Ошибка. Этот результат является диапазоном объединения множеств.
Option (A) $-1 \le x \le 4$ — Incorrect. This result is the range of the union of the sets.
选项 (B) $x > 0$ — 错误。忽略了集合 A 原本对变量上限的约束。
Вариант (B) $x > 0$ — Ошибка. Проигнорировано исходное ограничение множества A на верхний предел переменной.
Option (B) $x > 0$ — Incorrect. The original upper-limit constraint of set A was ignored.
选项 (C) $x \le 1$ — 错误。交集须同时满足集合 B 中的下限 $x>0$ 条件。
Вариант (C) $x \le 1$ — Ошибка. Пересечение должно одновременно удовлетворять условию нижней границы $x>0$ в множестве B.
Option (C) $x \le 1$ — Incorrect. The intersection must simultaneously satisfy the lower-limit condition $x>0$ from set B.
选项 (D) $0 < x \le 1$ — 正确。
Вариант (D) $0 < x \le 1$ — Верно.
Option (D) $0 < x \le 1$ — Correct.
TOPIC 07

二次不等式-大于0 (Квадратные неравенства: больше 0)

📐 公理法则 / THEOREM
当一元二次多项式 $ax^2+bx+c > 0$ （且 $a>0$）时，其解集为两根之“外”。
Когда квадратный многочлен $ax^2+bx+c > 0$ (при $a>0$), его множество решений находится «снаружи» от двух корней.
When a quadratic polynomial $ax^2+bx+c > 0$ (and $a>0$), its solution set lies "outside" the two roots.
💡 深度解析 / EXPLANATION
因为二次项系数为正，抛物线是个开口向上的“笑脸”。问我们“什么时候大于0”，其实就是在问图像的两边翘起来的触角对应什么 x 范围。所以答案是“取两边”（分散的区域：$x < 小根$ 或 $x > 大根$）。

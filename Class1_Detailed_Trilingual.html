Topic 01

集合基础 (Основы множеств) Основы множеств Bases of Sets

元素与集合的关系使用属于($\in$)与不属于($\notin$)符号表示。 Отношение между элементом и множеством обозначается символами принадлежности ($\in$) и непринадлежности ($\notin$). The relationship between an element and a set is denoted by the symbols of membership ($\in$) and non-membership ($\notin$).
你可以把集合想象成一个“箱子”，元素就是箱子里的“苹果”。谈论苹果在不在箱子里时，只能用“属于”这个词。注意区分 $\subseteq$（包含），那是用来描述“一个小箱子是否放在了大箱子里”。 Вы можете представить множество как «коробку», а элементы — как «яблоки» в ней. Когда мы говорим о том, находится ли яблоко в коробке, мы используем слово «принадлежит». Будьте осторожны, не путайте с $\subseteq$ (включает), которое описывает, «находится ли маленькая коробка внутри большой коробки». You can think of a set as a "box" and the elements as "apples" inside it. When talking about whether an apple is in the box, you can only use the word "belongs to". Distinguish this from $\subseteq$ (subset), which describes "whether a small box is placed inside a larger box".
Topic 02

子集判定 (Подмножества) Подмножества Subsets Determination

若集合A的所有元素都属于集合B，则称A是B的子集，记作 $A \subseteq B$。 Если все элементы множества A принадлежат множеству B, то A называется подмножеством B, что записывается как $A \subseteq B$. If all elements of set A belong to set B, then A is called a subset of B, denoted as $A \subseteq B$.
子集是集合与集合之间的关系。空集 $\emptyset$ 是任何集合的子集。含有特定元素的集合如 $\{a\}$ 本身是一个集合，所以它与另一个集合的关系必须用包含连词 $\subseteq$。 Подмножество — это отношение между двумя множествами. Пустое множество $\emptyset$ является подмножеством любого множества. Множество $\{a\}$, содержащее конкретный элемент, само по себе является множеством, поэтому его отношение к другому множеству должно выражаться символом включения $\subseteq$. A subset is a relationship between sets. The empty set $\emptyset$ is a subset of any set. A set containing specific elements, such as $\{a\}$, is itself a set, so its relationship with another set must use the inclusion symbol $\subseteq$.
Topic 03

并集计算 (Объединение множеств) Объединение множеств Union Calculation

由所有属于集合A或属于集合B的元素组成的集合，记作 $A \cup B$。 Множество, состоящее из всех элементов, которые принадлежат множеству A или множеству B, обозначается $A \cup B$. The set consisting of all elements that belong to set A or set B is denoted as $A \cup B$.
“并集”就是“合并”。在数轴上，把区间 A 和区间 B 画出来，只要有阴影覆盖的地方，全部合并到一起，取最左边的起点和最右边的终点。 «Объединение» означает «слияние». На числовой оси нарисуйте интервалы A и B. Вся заштрихованная область объединяется; просто возьмите самую левую начальную точку и самую правую конечную точку. "Union" means "merging". On the number line, draw interval A and interval B; wherever there is shaded coverage, merge them all together, taking the leftmost starting point and the rightmost ending point.
Topic 04

交集计算 (Пересечение множеств) Пересечение множеств Intersection Calculation

由属于A且同时属于B的元素构成的集合，记作 $A \cap B$。 Множество, состоящее из элементов, которые принадлежат и A, и B, обозначается $A \cap B$. The set consisting of elements that belong to A and simultaneously belong to B is denoted as $A \cap B$.
“交集”就是找“公共部分”。必须是 A 也有，且 B 也同时具备的范围。在数轴上画线，只有那段同时画了两层线的地方才是交集。 «Пересечение» означает поиск «общей части». Это то, что есть и в A, и в B. Если нарисовать линии на числовой оси, пересечением будет только тот отрезок, где проходят обе линии. "Intersection" means finding the "common part". It must be a range that exists in both A and B. When drawing lines on the number line, only the segment where both lines are drawn simultaneously is the intersection.
Topic 05

分式定义域 (Область определения дроби) Область определения дроби Domain of Fractions

对于含有未知数的分式表达式，分母不能等于0。 Для дробных выражений, содержащих неизвестные, знаменатель не может быть равен 0. For fractional expressions containing unknowns, the denominator cannot be equal to 0.
在数学界，0不能作为除数，这是最高铁律之一。当我们看到类似 $\frac{1}{x-a}$ 的函数时，第一反应必须是让下面的部分 $x-a \neq 0$维护表达式有意义。 В математике деление на 0 невозможно — это одно из самых строгих правил. Когда мы видим функцию вроде $\frac{1}{x-a}$, нашей первой реакцией должно быть обеспечение условия, чтобы знаменатель $x-a \neq 0$. In mathematics, 0 cannot be a divisor—this is one of the supreme iron rules. When we see a function like $\frac{1}{x-a}$, our first reaction must be to make the bottom part $x-a \neq 0$, ensuring the expression is meaningful.
Topic 06

根式定义域 (Область определения корня) Область определения корня Domain of Radicals

偶次根号内的代数式（被开方数）必须大于或等于0，即 $\ge 0$。 Алгебраическое выражение под корнем четной степени (подкоренное выражение) должно быть больше или равно 0, то есть $\ge 0$. The algebraic expression inside an even root (radicand) must be greater than or equal to 0, i.e., $\ge 0$.
在实数世界里，没有哪个数字平方后会等于负数，所以如果你要开平方（偶数次开方），里面的数绝对不能是负的。我们要求内部算式必须 $\ge 0$。如果根式还在分母上，那么连等于0都不行，只能 $>0$！ В мире действительных чисел нет числа, квадрат которого равен отрицательному числу. Поэтому при извлечении квадратного корня число внутри не может быть отрицательным. Мы требуем, чтобы оно было $\ge 0$. Если корень находится в знаменателе, он должен быть строго $>0$! In the world of real numbers, no number squared equals a negative number, so if you want to take a square root (even-numbered root), the number inside absolutely cannot be negative. We require the internal expression to be $\ge 0$. If the radical is in the denominator, then it can't even be 0; it must be $>0$!
Topic 07

二次不等式-大于0 (Квадратные неравенства: больше 0) Квадратные неравенства: больше 0 Quadratic Inequalities (>0)

当一元二次多项式 $ax^2+bx+c > 0$ （且 $a>0$）时，其解集为两根之“外”。 Когда квадратный многочлен $ax^2+bx+c > 0$ (при $a>0$), его множество решений находится «снаружи» от двух корней. When a quadratic polynomial $ax^2+bx+c > 0$ (and $a>0$), its solution set lies "outside" the two roots.
因为二次项系数为正，抛物线是个开口向上的“笑脸”。问我们“什么时候大于0”，其实就是在问图像的两边翘起来的触角对应什么 x 范围。所以答案是“取两边”（分散的区域：$x < 小根$ 或 $x > 大根$）。 Поскольку старший коэффициент положителен, парабола выглядит как «улыбающееся лицо» (ветви направлены вверх). Спросить «когда она больше 0» — значит спросить, где находятся ее поднимающиеся ветви по оси x. Ответ — «по краям» ($x < меньшего\ корня$ или $x > большего\ корня$). Since the quadratic coefficient is positive, the parabola is an upward-opening "smiley face". Asking us "when is it greater than 0" is actually asking what x-range corresponds to the upward-curling "antennas" on both sides of the graph. So the answer is "take both sides" (scattered regions: $x < smaller\ root$ or $x > larger\ root$).
Topic 08

二次不等式-小于0 (Квадратные неравенства: меньше 0) Квадратные неравенства: меньше 0 Quadratic Inequalities (<0)

当一元二次多项式 $ax^2+bx+c < 0$ （且 $a>0$）时，其解集为两根之“间”。 Когда квадратный многочлен $ax^2+bx+c < 0$ (при $a>0$), его множество решений находится «между» двумя корнями. When a quadratic polynomial $ax^2+bx+c < 0$ (and $a>0$), its solution set lies "between" the two roots.
同样的“笑脸”抛物线，当问我们 `<0` 小于零的区间时，我们在找它深陷在 x 轴下方那个兜底的部分。它就像一个陷入地下的坑，处于左右两个交叉点(根)的正中间，所以解集是连着的封闭区间。 В той же «улыбающейся параболе», когда нас спрашивают о диапазоне `<0`, мы проваливаемся в часть ниже оси x. Это похоже на яму под землей между двумя корнями, поэтому множество решений — это замкнутый интервал между ними. With the same "smiley face" parabola, when asked for the `<0` range, we are looking for the part deeply recessed below the x-axis. It's like a pit in the ground, located right in the middle of the two intersection points (roots), so the solution set is a continuous closed interval.

Q5
同步基础 / Basic
设集合 $A=[-5, 5], B=(0, 8]$，则 $A \cup B$ 的结果为： Пусть множества $A=[-5, 5], B=(0, 8]$, тогда результатом $A \cup B$ будет: Let $A=[-5, 5]$ and $B=(0, 8]$, the result of $A \cup B$ is:
A $0 < x \le 5$ $0 < x \le 5$ $0 < x \le 5$
B $x < 8$ $x < 8$ $x < 8$
C $-5 \le x \le 8$ $-5 \le x \le 8$ $-5 \le x \le 8$
D $x > -5$ $x > -5$ $x > -5$

Correct Answer: C

推演过程 / SOLUTIONS
**解答过程**：
求解**并集**即求两区间覆盖的所有范围。
根据题意，合并区域的左端点为 $-5$，右端点为 $8$。
综上求得范围：$-5 \le x \le 8$。
**Ход решения**:
Нахождение **объединения** означает поиск всего диапазона, охватываемого двумя интервалами.
Согласно условию задачи, левая конечная точка объединенной области равна $-5$, а правая конечная точка равна $8$.
В итоге получаем диапазон: $-5 \le x \le 8$.
**Solution process**:
Solving for the **union** means finding the entire range covered by both intervals.
The left endpoint of the combined region is $-5$, and the right endpoint is $8$.
Thus, the range is: $-5 \le x \le 8$.
选项剖析 / AUDIT
(A) $0 < x \le 5$ — 错误。此区间对应的是两集合的交集。 $0 < x \le 5$ — Ошибка. Этот интервал соответствует пересечению двух множеств. $0 < x \le 5$ — Incorrect. This interval corresponds to the intersection of the two sets.
